# AES Encrypter - Encryption Program

## Description

This project is an encryption program that uses the AES (Advanced Encryption Standard) algorithm to secure sensitive data. It enables users to encrypt and decrypt files or strings with a custom key, providing a straightforward and effective way to protect information.

## Features

- **AES Encryption**: Secure encryption using the AES algorithm.
- **User-Friendly Interface**: Simple command-line interface for ease of use.
- **File Support**: Ability to encrypt and decrypt various file types.
- **Key Generation**: Option to generate secure random keys.

## Usage

The program allows users to easily encrypt and decrypt data using their specified keys. Detailed instructions can be found in the documentation within the project.
